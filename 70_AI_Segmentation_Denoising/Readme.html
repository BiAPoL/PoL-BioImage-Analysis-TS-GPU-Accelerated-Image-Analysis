AI segmentation and denoising#

Welcome to the session on segmentation on segmentation and denoising using deep learning. This tutorial will pick up where the previous session on pytorch left off. This tutoorial will cover several topics:

  • Training a Unet for segmentation: The Unet was and is the working horse architecture for the segmentation of image data across all possible domains.

  • Denoising image data with Noise2Void: Noise2Void and its update, Probabilistic Noise2Void are powerful tools for denoising any image data without the need for ground truth data.

  • Denoising 3D image datawith Noise2Void: Noise2Void can be extended to 3D image data. This notebook will show you how to do that.
